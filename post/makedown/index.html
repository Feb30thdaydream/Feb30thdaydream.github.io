Makedown语法

2022-07-09 2 min read # study

1、一级标题
语法：“标题”+“===”（三个及以上，需要单独一行）
如：一级标题

2、二级标题
语法：标题”+“---”（三个及以上，需要单独一行）
如：二级标题

3、三级标题
不加任何符号就是三级标题

4、四级标题
语法：一个波浪线+内容+一个波浪线
如：内容

5、分割线“***”
语法：星号（三个及以上，需要单独一行）
如：

6、斜体
语法：一个星号+”内容“+一个星号
如：内容

7、粗体
语法：两个星号+”内容“+两个星号
如：内容

8、粗体加斜体
语法：三个星号+”内容“+三个星号
如：内容

9、标黄
语法：两个‘=’+“内容”+两个‘=’
如：内容

10、删除线
语法：两个波浪线+内容+两个波浪线
如：内容

11、无序列表
语法：加号、减号、乘号+空格（可多次叠加）+内容
注意：需要空一行才能跳出无序列表
如：

  • 一层
    • 二层
      • 三层

12、有序列表
语法：数字+‘.’+空格（可嵌套两层）
注意：需要空一行才能跳出有序列表
如：

  1. 内容
    1.1. 内容
    1.2. 内容
  2. 内容
  3. 内容

13、文字引用
语法：'>'+内容（可多层嵌套）
注意：需要空一行才能跳出文字引用
如：

第一层

第二层
第二层

第三层

14、行内代码块
语法：反引号+代码+反引号
如：Use the printf() function.

15、代码块
语法：三个反引号（独一行）+代码+三个反引号（独一行）
如：

#include<iostream>
int main(){
	printf("HelloWorld");
}

16、支持原生html语法
如：
橙色居右
